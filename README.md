## 🤔 Q&A

<b>Q1.</b> LMS, SELC 계정 정보를 저장하나요? <br>
<b>A1.</b> 한양대 노티파이어는 사용자의 LMS, SELC 계정 정보를 절대 저장하지 않습니다. 브라우저에 로그인되어 있는 세션 또는 쿠키 값을 취득하여 과목 정보 등을 조회합니다. 세션 또는 쿠키는 동기화 후 즉시 폐기되며 해당 값은 LMS, SELC 로그인때 마다 변경되는 동적인 값입니다. 

<br>

<b>Q2.</b> 자동 동기화 기능은 없나요? <br>
<b>A2.</b> 동기화는 수동으로만 가능합니다. 한양대 노티파이어는 사용자의 LMS, SELC 계정 정보를 저장하지 않으므로 자동 동기화가 불가합니다.

<br>

<b>Q3.</b> 어떤 종류의 푸시 알람을 지원하나요? <br> 
<b>Q3.</b> 현재는 이메일(SMTP) 푸시만 지원합니다. 이후 서비스 확장 시 디스코드, 슬렉, 카카오 등 푸시 확대 예정입니다.
